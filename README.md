# smart-feeder-server

API to control dog feeder device. Based on node, express and mongoose. 

## Install the dependencies
```bash
npm i
```

### Start the app in development mode (hot-code reloading, error reporting, etc.)
```bash
npm run dev
```

### Lint the files
```bash
npm run lint
```

### Build the app for production
```bash
npm run start
```
